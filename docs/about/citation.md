# Citation

If HarrisLabPlotting is part of your published work, please cite both the
package and the foundational reference for the seven-role classification
that underlies its modularity visualizations.

## Citing the package

```bibtex
@software{harrislabplotting,
  author       = {Harris Lab},
  title        = {HarrisLabPlotting: Brain connectivity and modularity visualization tools},
  year         = {2026},
  publisher    = {GitHub},
  url          = {https://github.com/AzadAzargushasb/HarrisLabPlotting},
  version      = {1.0.0}
}
```

Or in plain text:

> Harris Lab. *HarrisLabPlotting: Brain connectivity and modularity
> visualization tools* (Version 1.0.0). GitHub.
> <https://github.com/AzadAzargushasb/HarrisLabPlotting>

## Citing the seven-role classification

If you use the Guimerà–Amaral node roles (the colored borders in modular
plots), cite the original paper:

```bibtex
@article{guimera2005functional,
  author  = {Guimer{\`a}, Roger and {Nunes Amaral}, Lu{\'\i}s A.},
  title   = {Functional cartography of complex metabolic networks},
  journal = {Nature},
  volume  = {433},
  number  = {7028},
  pages   = {895--900},
  year    = {2005},
  doi     = {10.1038/nature03288}
}
```
