# Interactive plot gallery

Every plot kind the package can produce, with a static screenshot and the
live interactive figure side-by-side. Click the **Interactive** tab on
each block to load the actual 3D plot — rotate, zoom, hover for ROI
names, and toggle traces from the legend, exactly as you would in your
own output.

The HTML files are loaded from the docs site itself; no external
dependencies, no analytics. Each interactive embed is ~600 KB and
loads on demand.
